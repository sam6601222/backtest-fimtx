# FIMTX Backtesting Strategy
基於提供的 K 棒記錄圖進行策略回測。

## 策略定義
1. **週期定義**：
   - 大週期：60分K (60K)
   - 小週期：5分K (5K)
2. **均線表示**：(週期)/(MA均線)，例如 5/20 是 5分K的 20MA。
3. **回測目標**：
   - 條件：當 60/5 下彎（下行趨勢）時，5K 價格站上 5/20。
   - 動作：進場做多。
   - 計算：勝率與報酬。

## 專案結構
- `data_exploration/`: 分析圖片的原始記錄。
- `results/`: 回測結果摘要。
