# FIMTX Backtesting Strategy
基於提供的 K 棒記錄圖進行策略回測。

## 策略定義
1. **週期定義**：
   - 大週期：60分K (60K)
   - 小週期：5分K (5K)
2. **均線表示**：(週期)/(MA均線)，例如 5/20 是 5分K的 20MA。
3. **回測目標**：
   - 條件：當 60/5 下彎（下行趨勢）時，5K 價格站上 5/20。
   - 動作：進場做多。
4. **出場策略**：
   - 訊號 A：5/5 MA 下彎或價格跌破 5/5 MA。
   - 訊號 B：觸及 60K 關鍵壓力位（如 60/60 MA）。
   - 訊號 C：收盤（13:40）強迫出場。

## 專案結構
- `data_exploration/`: 分析圖片的原始記錄。
- `results/`: 回測結果摘要。
